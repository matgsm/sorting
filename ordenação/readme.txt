Novos arquivos podem ser adicionados pela pasta ./logs
O programa os detecta automaticamente e o lista entre as opções para o usuario escolher.
O arquivo sorting.py contem as estruturas de dados dos metodos de ordenação.
O arquivo Selects.py ontem as estruturas de dados necessárias para a seleção e leitura do arquivo a ser ordenados (.csv), bem como da seleção do metodo desejado para a ordenação
Para executar o programa basta executar o arquivo main.py
	em linux:
	python main.py
